Käytin kaikkia tarvitsimiani opittuja komentoja, joitakin useamman kerran (add, push, commit) kuin toisia (cherry-pick).

Ongelmien ilmetessä palasin teoriavideoihin, teoriaslideihin sekä pariin otteeseen kysyin GPT.ltä, että mikä on mennyt pieleen.

Ongelmat ratkesi teoriavideoiden ja slidejen avulla, ja muutamaan otteeseen GPT.n avulla.

Opin työstä käyttämään komentoja tehdessäni githubiin koodia, sinne koodin puskemista & pullaamista, branchien mergeemistä ja kaikki muutkin asiat, joita piti opetella tällä kurssilla.
